# Dataset

## Usage Instructions

1. Read two examples in folder /dataset_sample to understand format
2. Final dataset can be used for training located in /dataset, which include merged and unmerged version
3. Data cleaning and processing code are in /datacorrection

## Cautions

1. You must merge customers and transactions into one dataset before training
2. You can delete our data and replace them by your
3. Timestamp in merged data is Unix timestampt, which is good for ML but hard to read for human
